This is a modified OldStandard font with the nexts changes: the "onum" was modified for show more like at lining numbers, and was modified his kerning; was added a glyph called "NotCombiningDotBelow" in private area (U+E5CE) to using in papyrology; was replaced the own greek characters (greek aplá) for a modified version of GFS Porson, adquiring an appareance like typography used in the collection "Loeb Classical Library".
